Privacy info

To put it simply: I don't collect any data, but there are a few people involved who could.

I have no trackers on this page. No google analytics, no doubleclick, nothing. However, this page is hosted with GitHub Pages, who as far as I know don't do anything to the page, (except new sites will have a filter inserted to avoid people doing anything illegal) but they may collect your IP, which has location info. It refers to your wifi/ethernet network individually, and is unique to it. As far as I know, it's never used for anything other than to serve the right files to the right people, and the info is deleted soon after.

What every website gets by default:

  • Your IP
  • Location info from IP
  • ISP, based on IP

What is easy for websites to obtain:

  • Device characteristics, like screen size and density
  • UserAgent String, which contains info about your browser, OS, and versions of all
  • Through Google or another advertising company, they can get statistics about different users of the site, but Google uses this to build an advertising profile of those involved.
  • Google owns an advertising network that collects data about you and a website can place ads in the page for money. This site does not. Google will be quick to tell you that they don't sell your data, which is technically true, they sell ads that track you using their data that they "don't sell", in other words they sell your data.

Wanna learn webdev yourself? Here are some places to start:

  • Code.org - Learn basic HTML and other coding languages in the browser.
  • Codecademy - Another place to learn with many different courses.
  • W3Schools - A good reference for learning new HTML/CSS features and tags with an already established skillset. Also has samples of different designs with all the code included — Great for learning how to do very tough things like responsive navigation.
  • GitHub Pages - Free hosting for your projects, and a great way to learn Git. No serverside code though.
  • Atom - A free flexible extendable text editor. Fantastic auto-complete, (and if it doesn't work you can extend it with add-ons) Integrates directly into Git and GitHub for easy version control.
