Want VMF files?

I've thought about releasing VMF files for my maps/levels for a while, but I never had anyone that would host them, but github pages seems up to the task. Please note: Linux & Mac versions of the games CANNOT open VMFs, and they need to compile. These are not an alternate download.

If these open in plaintext, hit CTRL+S to save. Make sure that it saves as a VMF. If it doesn't, but saves as HTML, simply change .html to .vmf and it should work

gm_celltown

gm_celltown had an update a while ago that I am just now giving you access to as a VMF. This update added dynamic lights to certain objects. In the editor, these are env_projectedtextures, which GMod calls "lamps" ingame.

Download!

cp_transportation

cp_transportation was never finished. The idea is a TF2 map, but the attacking team (BLU) spawns in a vehicle. If I remember correctly, the layout is completely functional, but had no lighting or decoration, and I decided to scrap it partially because I didn't want to do lighting and partially because I didn't happen to have 32 people who could make sure it was a fun casual match.

Download!

phys_house_medium

phys_house_medium is a destructible house. It's not small but not big. It's also very hard to run because of dynamic lighting effects, which can be disabled. These are the same env_projectedtexture objects as before, except there are more rendered at once and I think they may be more complex with everything being 100% dynamic.

Download!

gm_beachplease

gm_beachplease will take longer than most maps to compile, as it uses high-quality shadows that are static. It's also the maximum size and has no fog, yet remains surprisingly performant.

Download!

Wanna learn webdev yourself? Here are some places to start:

  • Code.org - Learn basic HTML and other coding languages in the browser.
  • Codecademy - Another place to learn with many different courses.
  • W3Schools - A good reference for learning new HTML/CSS features and tags with an already established skillset. Also has samples of different designs with all the code included — Great for learning how to do very tough things like responsive navigation.
  • GitHub Pages - Free hosting for your projects, and a great way to learn Git. No serverside code though.
  • Atom - A free flexible extended text editor. Fantastic auto-complete, (and if it doesn't work you can extend it with add-ons) Integrates directly into Git and GitHub for easy version control.
